# Student-Registration-Form
This project is a simple Student Registration Form created using HTML and CSS. It allows students to enter their personal and academic details through a clean and user-friendly web form.

The form collects important information such as name, roll number, gender, email, mobile number, university, course, and address.

#Features:
Simple and responsive form design
User-friendly interface
Input validation using HTML required attribute
Dropdown selection for university and course
Gender selection using radio buttons
Clean layout using CSS styling

#Technologies Used:
HTML5
CSS3


#Form Fields Included:
Full Name
Roll Number
Gender
Email Address
Mobile Number
University Selection
Course Selection
Address
Submit Button
